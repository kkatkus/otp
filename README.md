<h1>React OTP component | <a target="_blank" href="https://otp.vercel.app/">storybook</a></h1>

<br />

To get it started, add `otp` to your project:

```sh
yarn add @kkatkus/otp
```

Or if you prefer using npm:

```sh
npm install @kkatkus/otp --save
```

<br />
<h1>How to use it</h1>

```ts
<OTP value={value} onChange={handleChange} />
```

 Example:
 ```ts
interface ExampleForm extends Record<string, UseFormValue> {
  email: string;
  password: string;
  repeatPassword: string;
  age: number;
};
```
